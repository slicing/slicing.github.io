java学习笔记（一）

2018-05-28

刚刚开始自己学习java的道路，很多细小的知识点需要积累，在根据视频学习的时候，总结了一些可能平常自己不会注意到的东西，但却非常重要，无论是对写代码还是打基础都至关重要。有什么不对的地方还请多多指教。

（1）主方法的作用是为了让函数运行起来，public static void main(String [] args)中，public：代表权限最大；void ：不需要返回值；static：静态，不需要对象，直接用给定的雷鸣就可以访问函数名；(String [] args)：主函数的参数列表，字符串属性类型的参数；args（1/2），arguments参数，该名称就是一个变量名。

（2）println：打印，在打印后面有一个回车（会产生换行），所以有时候需要使用print进行打印。

（3）一行注释：//；多行注释：/ /（多行注释里不能嵌入多行注释）；文档注释：/* /

（4）字符用‘ ’表示：‘1’，‘你’（这个也是正确的）；字符串用“ ”表示；

（5）数据类型，不仅仅有基本类型的八种（数值（int，short，long,byte,float,double）;字符（String）;布尔（boolean））;还有三种引用了类型：接口，对象，数组；在过大的数表示时要加上l。

（6）byte b=3;

b = b+1;    这个代码运行时错误的，等号右边的b是byte类型的，但所加的那个1是Int类型，不同类型不能相加。但如果等号右边是个常量就可以进行自动装换。

（7）& 两边都参与运算；&&若左边是false，右边不参与运算。

（8）a << b:表示 a*(2^b）优点：运算速度快，缺点：只能针对2的倍数。

a >> b  ：a/(2^b)  这样会在右移的时候产生高位空缺，因此原高位是什么，空的就补什么。（>>>）是无符号右移。

（9） x = (2 % 5) 2是被模数，5是模数 （a % b）

    a < b:     x = a;

    a = b:     x = 0;

若a<0;则x < 0;  

模2可以进行开关切换；

 任何；数模于模数必然小于模数。

（10）&：用来获取二进制某位的值。

^:一个数^这个数两次还是这个数。  6 ^ 3 ^ 3 = 6;

（11）switch(x)中语句无顺序。其中x可以是如下类型：byte,int short,char,enum,String;

（12）赋值运算符可以进行类型转换（自动转换）

short s = 4;

s = s + 5;(精度丢失）

s += 5;（没有丢失，自动装换）

（13）比较运算符：只有（true,false）两种结果

（14）逻辑运算符：用来连接boolean型表达式。
