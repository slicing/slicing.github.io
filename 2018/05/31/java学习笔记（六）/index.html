java学习笔记（六）

博主又要续了，emmmmm，

还是静态的，开场白。。

（1）静态变量：当该成员变量的值，每一个对象都一致时，就对该成员变量进行静态修饰。

（2）静态变量和成员变量的区别：

1.所属范围不同：  静态变量所属于类，也成为类变量

                            成员变量所属于对象，称为实例变量

2.调用不同：静态变量可以被对象和类调用（一般都用类名调用）

                    成员变量只能被对象调用

3.加载时期：静态变量随着类的加载而加载

                     成员变量随着对象加载而加载

  4.内存存储区域不同：静态变量存储在方法中

                                    成员变量存储在堆内存中

（3）静态代码块：随着类的加载而执行，仅执行一次

（4）静态变量有两次初始化，一次默认初始化，一次显示初始化。

补：构造函数的内部隐式部分：

1.super（），访问父类中的构造函数

2.成员变量的显示初始化

3.构造代码块初始化

4.构造函数内容自定义内容初始化
